Installation:

    Version 1 (VS Code Installation)

    1. Clone the repository
        git clone https://github.com/sjasthi/StyleX.git (Clone to repo)
        cd StyleX (Change into the project directory)

    2. Create a virtual environment

        Windows: 
            python -m venv env (create the virtual environment)
            env\Scripts\activate (activate the virtual environment)

        Mac/Linux:
            python -m venv env (create the virtual environment)
            source env/bin/activate (activate the virtual environment)

    3. Install the dependencies

        python -m pip install -r requirements.txt

    Version 2 (Google Colab - *This method of running the program is recommended*)

    1. Go to Google Colab
        https://colab.research.google.com
         Set your runtime type to GPU

    2. Clone the Repo (Run as one cell)

How to use the program:

    Generate a single image (VS Code): 

        Example: 
        
        python -m src.main --style anime --prompt "A cyberpunk samurai walking through neon Tokyo"

Batch prompts:
    1.   File format each prompt is separated by '#' (Note that you can use many '#' to separate each prompt not just one)

    Example:
    
        #
        Daytime cyberpunk city, futuristic megacity under bright daylight, towering high-tech skyscrapers, holographic advertisements visible in daylight, ultra detailed, wide angle view
        #
        Rain-soaked neon street at night, glowing pink and blue lights reflecting on wet pavement, cyberpunk marketplace, people with augmented reality visors, cinematic atmosphere
        #
        Cyberpunk samurai standing on a rooftop overlooking a futuristic skyline, glowing katana blade, neon signs in the distance, dramatic lighting, dynamic pose

        #
        Futuristic hacker workspace filled with holographic screens,
        floating UI elements, glowing keyboards, dark room illuminated 
        by blue neon light, detailed environment
        #

    This Prompt file has 4 different prompts which will be reader by the prompt_txt_file_reader.py file

    2. Place the file anywhere in the project
    3. Then run the program while being sure to include the path to the file in the command.

    Examples: 
        If your file is in a prompts folder (VS Code):
            .txt location: src/prompts/cyberpunk_batch_prompts.txt
            You would run this: python -m src.main --prompts-file src/prompts/cyberpunk_batch_prompts.txt --style cyberpunk 

    Google Colab:
        !python -m src.main --prompts-file src/prompts/cyberpunk_batch_prompts.txt --style cyberpunk 


Output:

    All generate images are saved automatically to: output_images/<style_name>/

    Example: 
        output_images/anime/
            20260305_154332_01.png
            20260305_154332_02.png

How Style Conditioning Works:

    Images from the style folder are loaded
    A subset of images is selected (aka a max of 10 images (by default) or whatever you specify) (NOTE: It is important to note that many images can cause undesirable outputs)
    Images are passed to the FLUX pipeline as conditioning inputs
    The model generates images influenced by those references.
